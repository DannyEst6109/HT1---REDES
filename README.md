## Descripción

Este repositorio contiene los entregables del Laboratorio 1. La **primera parte**
(reporte grupal) aborda los esquemas de comunicación Morse y Baudot, la transmisión
empaquetada y la conmutación de mensajes. La **segunda parte** (reportes individuales)
documenta la personalización de Wireshark, una captura con ring buffer y el análisis
de una transacción HTTP.
